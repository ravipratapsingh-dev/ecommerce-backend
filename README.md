# 🛒 E-commerce Backend (Spring Boot)

Backend system for E-commerce application built using Spring Boot.

## Features

- JWT Authentication
- User Login & Registration
- Product APIs
- Cart Management APIs
- Order Management APIs

## Tech Stack

- Java
- Spring Boot
- Spring Security (JWT)
- Hibernate / JPA
- MySQL

##  Frontend Repository

👉 https://github.com/ravipratapsingh-dev/ecommerce-frontend

##  Run Locally

mvn spring-boot:run
